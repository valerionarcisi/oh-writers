# Oh Writers — Specs

Le spec sono raggruppate in quattro epic. Ogni cartella ha un suo `README.md` con l'indice.

- **`core/`** — piattaforma + prodotto screenplay-first (no AI)
- **`ai/`** — feature che chiamano un LLM (generazione, assistenti, predizioni)
- **`infra/`** — deploy, design system, refactor interni, tooling
- **`agents/`** — interop esterna (MCP server, FDX/FCPXML, NLE). Placeholder.

La numerazione originaria dei file è stata mantenuta (es. `14-auto-logline-synopsis.md`) per non rompere i riferimenti testuali "Spec NN" sparsi nelle altre spec. I nuovi spec usano un numero progressivo coerente con l'epic che li contiene.

Vedi `CLAUDE.md` in questa cartella per le regole editoriali comuni.
